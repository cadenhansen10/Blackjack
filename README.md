# Blackjack Pro ♠️♥️♣️♦️

A professional-grade, cross-platform Blackjack game built with Python and Tkinter.

## Features
- **Casino-Style Interface**: Custom green felt table, dynamic card rendering, and professional poker chips.
- **Persistent Profiles**: Create a user profile to track your Win Rate and Net Earnings across sessions (stored locally).
- **Cross-Platform Design**: Custom UI elements ensuring consistent look and feel on macOS and Windows.
- **Smart Betting**: Input validation, balance tracking, and proper payout logic (3:2 for Blackjack).

## How to Run
No external installation required! This project bundles all necessary dependencies.

1. Clone the repository:
   ```bash
   git clone https://github.com/cadenhansen10/Blackjack.git
   ```
2. Navigate to the folder:
   ```bash
   cd Blackjack
   ```
3. Run the game:
   ```bash
   python3 Project13_gui.py
   ```

## Dependencies
- `tkinter` (Standard Python Library)
- `sqlite3` (Standard Python Library)
- `tkmacosx` (Bundled)
- `colour` (Bundled)
